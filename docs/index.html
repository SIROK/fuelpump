Button

UI Button

It is base of button. It is solid button. Please use button tag regardless it does not have a link. If it has a link, please specify role

Button 
<button class="btn" type="button">Button</button>
<a class="btn" href="#" role="button">Button</a>

Fill Button

<button class="btn -fill" type="button">Fill Button</button>
